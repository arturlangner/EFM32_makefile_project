# EFM32GG Makefile project
This is a basic, standalone Makefile project that can be compiled without Simplicity Studio.

It includes:
 - Build system based on boilermake
 - Linker script
 - Startup code
 - Header files
 - CMSIS headers
 - emlib
 - emdrv
 - SEGGER RTT
